算法整理（三）

发表于 2021-03-02 更新于 2021-03-03 分类于 算法

排序

  1. 冒泡排序：外循环0–n-1；内循环0–n-2-i；与下一位互换位置

  2. 快速排序：分治，关键点l,r，遍历，交换，递归处理左右两侧(l,j)与(j+1,r) 。 nlogn

  3. 归并排序：分治，关键点mid,l,r。先递归，再将左右两边的数组合并为一个数组，在赋值回原数组。即：先划分（递归进行划分），再排序 nlogn

    查找

  4. 二分查找\定界

    前提是待查找元素排列有序。时间复杂度由L降低为logL
